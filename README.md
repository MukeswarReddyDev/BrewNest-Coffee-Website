# ☕ BrewNest Cafe Website

A modern, responsive, and highly interactive web application designed for **BrewNest Cafe**, a premium specialty coffee roastery and café. The project showcases premium aesthetics, micro-animations, and interactive barista tools.

Live Preview available by opening the [index.html](index.html) file directly in your browser.

---

## 🌟 Key Features

*   **Interactive Brew Simulator**: Mix ingredients (Espresso, Steamed Milk, Foam, Hot Water, and Syrups) using responsive sliders to visualize the layers in a glass mug. The simulator dynamically recommends the matching menu item based on your ratios!
*   **Custom Drink Builder**: A detailed customization modal where customers can choose their milk base (Whole, Oat, Almond), sweetness levels, and additions (Extra espresso shot, whipped cream, caramel drizzle).
*   **Order Tray (Shopping Cart)**: A slide-out cart drawer showing order summaries, quantity selectors, applied promo code discounts (like `HAVEN20` or `BREW10`), and real-time subtotal/grand total calculation.
*   **Image Gallery Lightbox**: Interactive gallery displaying cafe culture, with a customized fullscreen lightbox viewer for keyboard/arrow-key navigation.
*   **Modern Design System**: Styled with a tailored premium dark-mode aesthetic (Coffee Bean Black, Gold, and Oat Milk Cream) utilizing CSS variables, responsive typography, and glassmorphic panels.

---

## 🛠️ Tech Stack

*   **HTML5 & CSS3**: Structured layout and custom styling with modern CSS variables.
*   **Bootstrap 5 & Icons**: Utilized for clean layouts, responsive grid alignments, modal components, and icons.
*   **Vanilla JavaScript (ES6)**: State management (for cart entries, pricing, and ingredients ratio) and UI animations.

---

## 🚀 How to Run Locally

1. Clone or download this project to your local computer.
2. Open the root [index.html](index.html) file in any modern web browser (Google Chrome, Microsoft Edge, Safari, Firefox).
3. Start configuring your custom drink and placing orders!
